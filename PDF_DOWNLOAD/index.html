At times, we might require the functionality of downloading a pdf on button click. This blog takes you through the key concepts and approach required to obtain the required functionality.

Click the button on the right to download the PDF

Code-Snippet

Add the following script from the source:
                        https://cdnjs.cloudflare.com/ajax/libs/jquery/3.1.0/jquery.min.js
            
     
 and then add the following code under script:
$('#GETFile').on('click', function(){
$.ajax({
                        url:'pdf_url',
                        method:'GET',
                                            xhrFields:{ responseType: 'blob'},
                                  success: function(data){
                                                        var a = document.createElement('a');
                                                               var url = window.URL.createObjectURL(data);
                                    a.href = url;
                                            a.download = 'f.pdf';
                                            document.body.append(a);
                                a.click();
                                a.remove();
                                                    window.URL.revokeObjectURL(url)
            }
    });
});
